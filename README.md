## Instalação 
**Linguagem**: C/c++<br>
**Framework**: C++ STL (Standard Template Library)<br>

Ter o GCC (GNU C Compiler) e o G++ (GNU C++ Compiler) ambos instalados

## Uso 
Para compilar código C : `gcc -Wall [arquivo.c] -o out`


Para compilar código C++ : `g++ [arquivo.cpp] -o out`


Para executar: ./out
